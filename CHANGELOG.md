# Changelog

## 1.0.0 - 2026-06-09

### Added

- Solidity `CollateralVault` with collateral deposits, insurance liquidity, generic settlement, guarded withdrawals, and settlement IDs.
- Local/test `MockUSDC` collateral token.
- Fastify/TypeScript gateway API.
- EIP-712 `SignedIntent` verification.
- App registry with `X-App-Id` and `X-App-Secret` authorization.
- Admin/operator bearer authorization.
- Generic settlement API with linked signed intents.
- Settlement audit reports.
- User-signed withdrawal requests.
- Blockchain event indexer.
- Reconciliation service.
- WebSocket event stream.
- Swagger/OpenAPI documentation.
- Static Operator Console.
- Trading reference application.
- Standalone external-client integration example.
- Contract, backend, and end-to-end tests.
- GitHub-ready documentation and CI workflow.
