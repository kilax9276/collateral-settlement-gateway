# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different experience levels and viewpoints.
- Provide constructive feedback.
- Focus on the project and technical substance.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Public disclosure of private security issues.
- Posting secrets, private keys, or sensitive infrastructure details.
- Spam or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct.
